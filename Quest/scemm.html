Opening Procedures‌

Daily opening procedures for the Clover Vertica.


Prerequisites

Materials needed:

  • Coffee beans

  • Hoppers

  • Markout pitchers


Context

Frequency: Daily

  1. Press start button to begin opening procedures


    NOTE: You will not be able to begin if all hoppers are not in place on top of the machine.


    • Select anywhere on the touchscreen to wake up if needed.

    • Remove drip trays and ensure markout pitchers are below both brewer spouts.


  2. Initiate Rinse Cycle


    • Two rinse cycles will complete.

    • Ensure markout pitchers are both in place.

    • Once rinse cycle is complete, empty pitchers into sink and place them back under the brew spouts.


  3. Fill each hopper with the designated coffee beans (if not already filled and left overnight)



    • Ensure matching hopper signage is in place.


  4. Press Next to assign bean types.


    • Follow the Assign Bean Type steps for detailed instructions


  5. Press Start Seasoning to brew seasoning cups


    • Machine will pull beans from all three hoppers to season the chamber in one brew to season the filter

    • Ensure markout pitchers are below brew spouts

    • Once Seasoning is complete, empty pitchers into sink and replace drip trays.

Assign Bean Types‌‌

Procedure for changing and assigning bean types as needed.


  1. Select ‘Bean Recipes’ to view menu


  2. Choose hopper 1-6 for bean type


    • Hoppers can be assigned individually by selecting the number from left to right, 1-6


  3. Indicate bean roast type


    • Select Blonde, Medium, Dark or Reserve based upon the bean type


  4. Select name of desired bean


    • Bean type will be specific to coffee name

      NOTE: Select name, if Reserve is selected



  5. Select 'More' to view additional bean types if the one needed is not listed on the screen

  6. Repeat steps 2-5 as needed to ensure all bean hoppers have the correct recipe


    NOTE: Iced Coffee will be under Medium roast

    Run Cleaning Cycle‌

    Daily, or when needed, cleaning cycle procedures for the Clover Vertica.


    Prerequisites

    Materials needed:

    • Wiping cloth

    • Machine Warewash Detergent

    • Rubber gloves

    • Splash goggles (as needed)


      SDS:Safety Data Sheets for the following are available on Partner Hub or StoreLink

    • QSR Heavy Duty Multi-Purpose Sink Detergent

    • ClickSan® Disinfectant/Sanitizer / KAY-5® Sanitizer

    • Sani Wipes

    • Machine Warewash Detergent


Context

Frequency: Daily, end of day, or when prompted


A cleaning cycle shall be performed every 24 hours while in use. If the machine is in-use for 26 hours without a cleaning cycle, then the machine will stop operating and only completing a cleaning cycle will

allow for coffee to be brewed.


  1. Select Cleaning to begin

    • Touch anywhere on the touchscreen to see menu


  2. Follow prompts and all cleaning steps as outlined on the touchscreen

    • Once you start the cleaning cycle by pressing ‘Acknowledge’ you will not be able to end until the full cleaning is complete (~15 min)


      NOTE: Failure to follow all steps thoroughly will result in machine malfunction.


  3. Remove drip trays & backsplashes

    • All parts are machine washable


  4. Remove brew modules

    • Lift button panel and pull out brew module by black handle


  5. Remove wipers

    • Lift up on small metal wire to unlock, then pull out wiper

    • Wipers arms will retract after closing button panel. Do not open the button panel while wiper arms retract.


  6. Open button panels and wipe interior with damp sanitized cloth

    • Make sure to concentrate on cleaning the filter and all interior surfaces of dry coffee grounds


  7. Remove grounds drawers

    • Remove grounds drawer and wash all components in dish machine


      NOTE: Do not turn off the machine. It will automatically enter a sleep mode after a set time of non-use.


  8. Replace wipers

    • Ensure the wipers are facing frontwards and the metal bar is locked down in place. Do not open button panel while wipers retract


  9. After wipers retract, add Urnex cleaning tablets

    • Add three tablets into each chamber


10 Replace brew modules, grounds drawers & backsplashes

. • All parts dishwasher cleaned


  1. Start Soak/Rinse Cycle

    • Soak Cycle will last 11 minutes. Then two rinse cycles will run lasting 4 minutes and 1 minute each

    • A countdown timer will display remaining time

    • Ensure both pour spouts have a markout pitcher underneath them throughout the 3 soak/rinse cycles

    • Cleaning chemical rinse solution collected in markout pitchers can be disposed of in a brew station dump sink or the three-compartment sink, not to be disposed of in a hand sink or in-use utensil rinse sinks (which are considered food prep sinks)


NOTE: Soak/Rinse Cycle Machine time should be used to clean the exterior of the machine and hopper fixture. See Cleaning the Machine Exterior and hopper fixture in the next section.

Machine Exterior‌‌

Procedure for the end of day cleaning of the machine exterior, hopper fixer and bean hoppers


Prerequisites

Materials needed:

  • ClickSan® Disinfectant/Sanitizer / KAY-5® Sanitizer

  • QSR Heavy Duty Multi-Purpose Sink Detergent

  • Wiping cloth

  • Machine Warewash Detergent

  • Rubber gloves

  • Splash goggles (as needed)


    SDS:Safety Data Sheets for the following are available on Partner Hub or StoreLink

  • QSR Heavy Duty Multi-Purpose Sink Detergent

  • ClickSan® Disinfectant/Sanitizer / KAY-5® Sanitizer

  • Sani Wipes

  • Machine Warewash Detergent


Context

Frequency: Daily at end of day

  1. Clean exterior surfaces


    • Use a wiping cloth saturated in sanitizer

    • Remove all coffee residue


  2. Remove hoppers



    • Gently push back and push up to remove


  3. Remove bean funnels


    • Remove both bean funnels and wash in dishwasher. Allow to air dry.


  4. Vacuum all bean debris from machine top and underneath bean funnels


  5. Replace all components

    • Beans should be stored in hoppers on the machine overnight


NOTE: DO NOT replace bean funnel or hopper components that are still wet to the machine. This will cause grinders to seize up and the machine to malfunction.
